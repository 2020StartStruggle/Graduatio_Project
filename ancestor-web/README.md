# 家谱前台

#### 介绍
准备把我前后做了很多年的家谱程序整理出来，分享给大家。
儿时，总在问家谱，爷爷叫什么，老太爷叫什么，老老太爷叫什么，一般能知道老太爷叫什么差不多了。
后来偶然机会得到了老家谱，于是前后花了近十年时间来修家谱，目前基本上修完了。
如果你也想做类似的事情，希望可以帮到你。
